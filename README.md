# Geometric Calculator (C++)

## Overview

Geometric Calculator is a Programming Fundamentals course project developed in C++ using Dev-C++.

The application provides a menu-driven interface for calculating properties of multiple 2D and 3D geometric shapes. It allows users to select a shape, enter the required dimensions, and calculate measurements such as area, perimeter, circumference, surface area, and volume.

---

- Square

- Parallelogram
- Trapezoid
- Hexagon

### 3D Shapes

- Sphere

- Cuboid
